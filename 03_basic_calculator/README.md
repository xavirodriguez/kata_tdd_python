# Basic Calculator Kata

Create a simple calculator that supports four operations.

## Features

- `add(a, b)`
- `subtract(a, b)`
- `multiply(a, b)`
- `divide(a, b)` → Raise error if dividing by zero.

## Example

```python
add(2, 3)       # → 5
divide(10, 2)   # → 5.0
divide(4, 0)    # → ZeroDivisionError
